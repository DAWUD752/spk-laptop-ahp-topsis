# SPK Pemilihan Laptop Terbaik
Metode AHP dan TOPSIS menggunakan Python dan Streamlit.

## Kriteria
- Price (Cost)
- RAM (Benefit)
- Storage (Benefit)
- CPU_Speed (Benefit)
- Weight (Cost)

## Metode
1. AHP untuk menentukan bobot kriteria
2. TOPSIS untuk perankingan alternatif laptop

## Cara Menjalankan
```bash
pip install -r requirements.txt
streamlit run app.py
